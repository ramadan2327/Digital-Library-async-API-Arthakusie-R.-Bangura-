# Digital Library Async API Simulation

This project is a simple asynchronous digital library system implemented using Python. It simulates basic API functionality such as borrowing and returning books and demonstrates how multiple users can access the system concurrently.

## Features
- Asynchronous programming using async and await
- Simulated API endpoints for borrowing and returning books
- Support for multiple concurrent users
- Prevention of race conditions using asyncio.Lock

## Technologies Used
- Python
- asyncio

## How to Run the Project
1. Ensure Python 3 is installed
2. Download or clone the repository
3. Run the program using the command:
   python main.py
